# Trabajo-mushroom-grupal

## Descripción

Este repositorio contiene un análisis estadístico del conjunto de datos *Mushroom Classification*,
realizado mediante R y R Markdown. El objetivo del estudio es evaluar la relación entre características
categóricas de los hongos (olor y magulladuras) y su toxicidad (comestible o venenosa).

## Estructura del repositorio

- `index.html`: versión web del informe
- `Mushrroomm.Rmd`: documento R Markdown con el código y análisis
- `estilo.css`: hoja de estilos del documento
- `bibliografia.bib`: referencias bibliográficas
- `libraries.bib`: referencias de paquetes de R
- `images/`: imágenes utilizadas en el informe

## Disclaimer

Usamos librerias como ggplot, dplyer
