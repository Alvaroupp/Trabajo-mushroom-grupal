## Descripción

Este repositorio contiene un análisis estadístico del conjunto de datos *Mushroom Classification*,
realizado mediante R y R Markdown. El objetivo del estudio es evaluar la relación entre características
categóricas de los hongos (olor y magulladuras) y su toxicidad (comestible o venenosa).

## Estructura del repositorio

- `index.html`: versión web del informe
- `Mushrroomm.Rmd`: documento R Markdown con el código y análisis
- `estilo.css`: hoja de estilos del documento
- `bibliografia.bib`: referencias bibliográficas
- `libraries.bib`: referencias de paquetes de R
- `images/`: imágenes utilizadas en el informe

## Disclaimer

Este trabajo ha sido desarrollado utilizando el entorno de programación R y el IDE RStudio, empleando diversas librerías de código abierto ampliamente utilizadas en el análisis y visualización de datos. En particular, se han utilizado las siguientes:

**dplyr**: para la manipulación y transformación de datos, incluyendo selección de variables, recodificación y creación de nuevas variables categóricas.

**ggplot2**: para la generación de representaciones gráficas, tales como gráficos de barras, permitiendo una visualización clara de la relación entre variables categóricas.

**knitr**: para la generación de tablas y la integración de código R dentro del documento R Markdown.

**broom**: para la extracción y presentación ordenada de los resultados de los tests estadísticos, facilitando su interpretación y presentación en formato tabular.

Todas las librerías utilizadas son de libre acceso y forman parte del ecosistema estándar de análisis de datos en R. El uso de estas herramientas se ha realizado exclusivamente con fines académicos y educativos.
